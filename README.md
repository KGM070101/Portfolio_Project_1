# Portfolio Project

## 프로젝트 개요

하늘에서 떨어지는 낙하물을 피하면서 점수를 획득하는 2D 생존게임이다.  
플레이어는 다양한 낙하물과 아이템이 동시에 등장하는 상황에서 생존하며, 더 높은 점수를 목표로 플레이한다.

시간이 지날수록 낙하물의 수가 증가하고 새로운 종류의 낙하물이 등장하여 난이도가 점점 상승한다.

---

## 게임 구조

- 플레이어는 낙하물을 피하며 생존한다.
- 낙하물에 맞으면 체력이 감소한다.
- 아이템을 획득하면 생존력 또는 점수 획득 효율이 상승한다.
- 점수가 오를수록 새로운 낙하물이 등장하고 게임이 더 어려워진다.

---

## 낙하물 종류

| 낙하물 | 특징 | 효과 |
|--------|------|------|
| 브로콜리 | 가장 기본이 되는 낙하물, 속도가 느리고 변칙성이 적음 | HP -1 |
| 토마토 | 브로콜리보다 빠르고 변칙성이 큼, 등장 수는 더 적음 | HP -1 |
| 수박 | 크기가 가장 크며, 땅에 떨어질 때 한 번 튕긴 뒤 사라짐 | HP -2 |
| 와사비 | 플레이어에게 직접 피해를 주지는 않지만, 땅에 닿으면 일정 범위를 끈끈하게 만듦 | HP -0 |

---

## 아이템 종류 

| 아이템 | 효과 | 지속 시간 |
|--------|------|-----------|
| ShieldItem | 하늘에서 떨어지는 물체를 막아주는 방어막 생성 | 5초 |
| LifeItem | 체력 1 증가 | 즉시 적용 |
| SpeedItem | 이동 속도 2배 증가 | 8초 |
| DoubleScoreItem | 획득 점수 2배 | 5초 |
| SmallItem | 플레이어 크기 1/2 감소 | 8초 |
| LiskItem | 낙하물 속도 2배, 대신 획득 점수 4배 | 5초 |
| SamuraiItem | 낙하물과 충돌 시 체력이 닳지 않고 낙하물을 제거, 제거한 아이템(브로콜리, 토마토) = 5점 (수박) = 10점, 이동속도 증가 + 2단점프 가능 + 대쉬 거리 증가 등등, 기동성 증가| 10초 |

| 아이템 | 확률 |
|--------|------|
| ShieldItem | 20% |

| SpeedItem | 20% |
| SmallItem | 20% |
| DoubleScoreItem | 10% |
| LiskItem | 5% |
| SamuraiItem | 5% |

---

## 레벨 시스템 

| 점수 구간 | 변화 |
|----------|------|
| 0 ~ 100 | 브로콜리만 등장, 낙하물 수 적음 |
| 100 ~ 250 | 토마토 등장, 브로콜리 수 증가 |
| 250 ~ 400 | 브로콜리와 토마토의 기본 수 증가, 토마토 변칙성 증가,브로콜리 변칙성 증가 |
| 400 ~ 700 | 수박 등장 |
| 700 ~ 1000 | 와사비 등장 |
| 1000 ~ ∞ | 브로콜리 제외 낙하물 수 증가(브로콜리는 변칙성 증가), 수박은 두번 튕김(튕긴 후 두 조각으로 분열) |

---

## 오브젝트 애니메이션

### 아이템
- 구체 형태로 회전하며 내려오는 애니메이션

### 수박
- 1번 충돌시 균열, 2번 충돌시 2개로 쪼개지는 애니메이션

### 와사비
- 땅에 닿으면 퍼지는 애니메이션

### 사무라이 상태
- 사무라이 아이템 획득 시 칼을 든 상태로 변경
- 낙하물과 충돌할 때마다 칼로 튕겨내는 애니메이션

### 스피드 상태
- 달리기 애니메이션으로 변경
